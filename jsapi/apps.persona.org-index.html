This site hosts the Javascript implementation of the Mozilla Marketplace Apps Javascript API. This provides the navigator.mozApps API to browsers that do not natively support it.
To use this library, include this in your page: 
 <script src="https://myapps.mozillalabs.com/jsapi/include.js"></script>
this will not override any browser-native implementation of navigator.mozApps
